Move it in

The Simple Data Pipe moves JSON from web services like Salesforce.com and Stripe, and captures it in Cloudant. Once persisted in Cloudant, you can do all kinds of different stuff.

  • First time here? Pause now, and go skim our tutorial on connecting to Salesforce.com before piping your first data. It will provide an overview of the mechanics of The Pipe, even if you don't need visualizations just yet.
  • Start fast: Choose "Create A New Pipe" and use one of the built-in options found under Type. We're always looking to add more. Check our full list of connector options for updates.
  • BYOP: Build your own pipe with the JavaScript framework we provide. If you build something cool and want to see it become part of a future Simple Data Pipe, let us know. Contributions are always welcome!

Move it out

Moving data out of Cloudant is just as easy. The database comes with free, native integrations for loading JSON to data warehousing and analytics engines. From there, you're just a step away from the reporting and visualization tools you know and love.

Use the Cloudant-dashDB integration to sync data into IBM's cloud data warehouse service. There's the Cloudant-Apache Spark integration for those who want to explore JSON in Jupyter notebooks, or break all the rules by using Spark SQL to query "unstructured" JSON. The open-source Transporter ETL pipeline from Compose.io is a great choice for moving between a range of non-relational data stores like MongoDB, Redis, and Elasticsearch â€” all of which are, coincidentally, available on IBM's Compose service.

Need to go the enterprise route? We have you covered with IBM's DataWorks data shaping and movement API. It's a paid service, so you'll need to enter credit card info on Bluemix before provisioning.

Not ready to leave Cloudant just yet? Visualize JSON right from the database by building secondary indexes to organize data that feeds into visualizations in the D3.js library.
