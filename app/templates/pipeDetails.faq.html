FAQs

Where can I learn more about the Simple Data Pipe?

Check our resources page for all things Data Pipe. We keep it updated with tutorials and new connectors.

I found a bug. Where should I report it?

Please submit an issue on Github and we'll get right on it.

Are there limits the amount of data I can move?

No limits we're aware of. We tested up to 100,000 Salesforce.com records. If you pass that threshold and have problems, submit an issue.

Can I contribute?

Yes, of course. The Simple Data Pipe is open source! We have lots of features planned, including more connectors. Have ideas? We'd love your help. Head over to Github and get your pipe on.

How can I build a connection to a Web service?

We provide a JavaScript framework to help you connect your own Web services to the Simple Data Pipe. Before digging in, be sure to check our growing library of prebuilt connectors. We might already have the connection you need.

How can I move data from Cloudant to another database or warehouse?

There are several options:

  • Out-of-the-box, you can sync data into IBM's dashDB cloud data warehouse. Launch the Cloudant dashboard from the sdp-cloudant-service service tile in your Bluemix application, and the Warehousing tab in the Cloudant UI will walk you through the rest.
  • Up for a little config, for a lot of flexibility? The open-source Transporter ETL pipeline from Compose.io moves data between a range of non-relational data stores like MongoDB, Redis, and Elasticsearch.
  • Need full API access? We have you covered with IBM's DataWorks data shaping and movement API. It's a paid service, so you'll need to enter credit card info on Bluemix before provisioning.

How can I analyze the data in Cloudant?

For basic aggregations, you can visualize JSON right out of Cloudant via D3.js. For heavier-duty data analysis, the Cloudant-Apache Spark integration lets you explore JSON in Jupyter interactive notebooks. You'll need to provision the Apache Spark service on Bluemix, but from there you'll have access to Python, Scala, and Spark SQL to analyze data until your heart's content.

Who is behind the Simple Data Pipe?

Sarah Maston and David Taieb from the IBM Cloud Data Services developer advocacy team spearheaded this project, and are driving it forward. There are lots of ways to connect with them, if you'd like to learn more.
